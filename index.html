array.filter( )

  1. Toma un array
  2. Una funcion de callback filtra el array
  3. El filtro es aplicado a cada item del array
  4. Si el valor matchea, es a√±anido a un nuevo array
  5. Si el valor NO matchea, es ignorado
  6. Retorna un nuevo array

Ejemplo

JavaScript

array.filter( )

  • var fruits = ['pear', 'banana', 'plum'];
  • Fruits contains an array with 3 strings
  • const result = fruits.filter(fruit => fruit.length < 5);
  • Fruit represents each array item
  • console.log(result);
  • Values less than 5 characters are added to a new array and stored in result
  • // output: Array ['pear', 'plum']
